This repository hosts a notebook that fetches glider data/metadata from https://gliders.ioos.us/erddap and some compute metrics.

To run the notebook on binder click the binder badge below,

[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/ocefpaf/glider_metrics/HEAD?labpath=glider_metrics.ipynb)

or try it on Google Colab.

<a target="_blank" href="https://colab.research.google.com/github/ocefpaf/glider_metrics/blob/main/glider_metrics.ipynb">
  <img src="https://colab.research.google.com/assets/colab-badge.svg" alt="Open In Colab"/>
</a>

One can modify the query parameters in the second cell of the notebook.

If you wish to clone and install this locally please:

1. Install the IOOS environment following https://ioos.github.io/ioos_code_lab/content/ioos_installation_conda.html.
2. Clone this repository or download it as a zip file using the GitHub interface.
3. Fire up jupyter notebook (or jupyterlab), modify the query and run all the cells.

If you find any problems please [open an issue](https://github.com/ocefpaf/glider_metrics/issues) and let us know!
